O que é Git?

Git é um sistema de controle de versão que ajuda a organizar o histórico de alterações no código.

O que é GitHub?

GitHub é uma plataforma online que usa Git para armazenar e compartilhar projetos.

Comandos Git

Comando Descrição
git init Inicia um repositório Git
git add . Adiciona arquivos para o commit
git commit -m "mensagem" Salva as alterações com uma mensagem
git push Envia alterações para o repositório remoto
git status Mostra o estado atual do repositório
git log Exibe o histórico de commits
git clone Copia um repositório existente
git pull Atualiza o repositório local com o remoto
git branch Lista ou cria branches
git checkout Troca de branch
